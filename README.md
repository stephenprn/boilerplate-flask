# Flask SQLAlchemy API boilerplate

## Development set-up

1. Create virtual environment, source it and install dev dependencies
```
make init_dev
```

2. Start docker db and start local server
```
make start_dev
```

3. Install pre-commit hooks
```
pre-commit install
```
